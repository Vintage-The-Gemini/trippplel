## Project Overview

This is **TrippleLuxe** — a Next.js + Express e-commerce store selling t-shirts and hoodies in Kenya (prices in KES).

- **Frontend:** Next.js 16 (App Router, TypeScript, Tailwind CSS) — `Trippplel/client/`
- **Backend:** Express.js with Prisma ORM — `Trippplel/server/`
- **Database:** Supabase PostgreSQL
- **Payments:** Pesapal (M-Pesa + cards)
- **Images:** Cloudinary
- **Hosting:** Render.com
